# Security

This is a personal sample repository, not a supported product. There is
no formal vulnerability disclosure program.

If you find a security issue in this code (e.g. a Bicep pattern that
unintentionally exposes a resource, a sample command that leaks
credentials, or a doc that recommends an unsafe configuration), please:

1. **Do not** open a public GitHub issue with reproduction details.
2. Open a minimal public issue noting *"security-sensitive — please
   contact me for details"*, **or** reach out via the GitHub profile at
   [@dbarkol](https://github.com/dbarkol).
3. I'll respond and coordinate a fix on a private branch before any
   public disclosure.

For vulnerabilities in the underlying Azure services (Azure Monitor,
Application Insights, Azure AI Foundry, Bastion, Private Link, etc.),
please report them directly to the **Microsoft Security Response Center
(MSRC)** at <https://msrc.microsoft.com/>.
